Main Game Spoilers for: NTWEWY.

Added in the vanishing intro colour for each day.

There is a cycle from red to purple, blue, green, yellow, orange, and grey. A "reverse" rainbow with grey added in.

It's interesting to see that colours appear in the same order. When we reach "The Last day", the seventh day, is coloured grey as expected. The next "day", "The Last day'" is also grey while "The Last day''" is red, the next colour in the cycle, which could indicate a day has passed. Or more.

They ended up becoming an influence for Father's gloves.
